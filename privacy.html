🌑 ぽもじかん ── プライバシーポリシー

最終更新：2026年5月12日 (v2.9)

1. 収集する情報

本アプリは、いかなる個人情報も収集・送信しません。

すべてのデータ（完走数・コレクション・パーティ・設定・統計）は、お使いの端末の localStorage に保存され、外部に送信されることはありません。

2. 端末に保存するデータ

  • タイマー設定（作業/休憩の分数・秒数・言葉モード）
  • 完走数（今日 / 累計 / 日次・時間帯）
  • 連続日数（ストリーク）・達成バッジ・デイリーチャレンジ進捗
  • ぽもじ図鑑（落ちた文字の出現回数・カテゴリ・☆ レア情報）
  • パーティ情報（主人公・仲間 3 体）・解放級 (雨/泡 別)
  • 音・バイブ・通知・テーマ・フォント・パレットの設定
  • 自動バックアップ（直近 3 世代のスナップショット）

これらは端末を変えると引き継がれません（メニューの「データ書き出し / 取り込み」で手動移行できます）。

3. 通知の利用

通知許可を与えた場合、タイマー完了時にローカル通知（Notifications API）を表示します。通知内容は端末内で生成され、外部に送られません。

4. 第三者サービス

  • GitHub Pages：本アプリのホスティング元です。アクセス時に GitHub の利用規約・プライバシーポリシーが適用されます。
  • Google Fonts：書体（Noto Serif JP / JetBrains Mono）をネットワーク経由で取得します。Google のフォントサービスに依存します。

本アプリ自体は、アクセス解析・広告・トラッキング Cookie を一切使用していません。

5. データの削除

メニュー →「💾 データ」→「📚 図鑑リセット」または「🌑 全リセット」でいつでもすべての記録を削除できます。ブラウザの設定からサイトデータを削除しても全データが消えます。

6. 子どもの利用

本アプリは全年齢向けです。13歳未満のお子様の利用も保護者の判断で問題ありません。

7. ポリシーの変更

本ポリシーが更新された場合、本ページの「最終更新」日付を変更してお知らせします。

8. お問い合わせ

本アプリ・ポリシーに関するお問い合わせは GitHub Issues までお願いいたします。
