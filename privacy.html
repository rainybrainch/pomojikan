🌑 ぽもじかん ── プライバシーポリシー

最終更新：2026年5月18日 (v10n15)

1. 収集する情報

本アプリは、いかなる個人情報も収集・送信しません。

すべてのデータ（完走数・コレクション・パーティ・設定・統計）は、お使いの端末の localStorage（キー名 pomojikan_v30）に保存され、外部に送信されることはありません。

2. 端末に保存するデータ

  • タイマー設定（作業/休憩の分数・秒数・プリセット）
  • 完走サイクル数（累計）
  • ぽもじ図鑑（出会った字の出現回数・タグ・★ レア情報）
  • パーティ情報（リーダー・仲間 最大 3 体・各字の Lv / EXP / パーク）
  • 解放済みレア帯（★1〜★6）・永続バフ
  • 音響・オンボーディング完了フラグ
  • ユーザー ID（P-XXXX-XXXX 形式・初回起動時に端末内で自動発行）

これらは端末を変えると引き継がれません。📊 記録 →「引継ぎコード」を発行 → 別端末で復元、の手順で手動移行できます（コードはあなたが管理し、サーバーには送られません）。

3. 通知の利用

通知許可を与えた場合、タイマー完了時にローカル通知（Notifications API）を表示します。通知内容は端末内で生成され、外部に送られません。

3.1 画面スリープ抑制（Wake Lock）

作業中／休憩中のみ画面が暗くなるのを抑制します（Screen Wake Lock API）。停止時に自動的に解放されます。データ送信は行いません。

3.2 ピクチャー・イン・ピクチャー（PiP）

メニュー「📺 小窓タイマー」を明示的にタップした時のみ、ブラウザ標準の Document Picture-in-Picture API を使って小窓タイマーを表示します（Chrome / Edge デスクトップ限定）。データ送信は行いません。

4. 第三者サービス

  • GitHub Pages：本アプリのホスティング元です。アクセス時に GitHub の利用規約・プライバシーポリシーが適用されます。
  • Google Fonts：書体（Noto Serif JP / JetBrains Mono）をネットワーク経由で取得します。Google のフォントサービスに依存します。

本アプリ自体は、アクセス解析・広告・トラッキング Cookie を一切使用していません。

5. データの削除

📊 記録 →「危険な操作」→「全データを消去」（2 段階確認）でいつでもすべての記録を削除できます。ブラウザの設定からサイトデータを削除しても全データが消えます。

6. 子どもの利用

本アプリは全年齢向けです。13歳未満のお子様の利用も保護者の判断で問題ありません。

7. ポリシーの変更

本ポリシーが更新された場合、本ページの「最終更新」日付を変更してお知らせします。

8. お問い合わせ

本アプリ・ポリシーに関するお問い合わせは GitHub Issues までお願いいたします。
