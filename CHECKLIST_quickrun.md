# 🏃 ぽもじかん 1 サイクル 実機 クイックチェック（v10n19）

> 25 分作業 + 5 分休憩を 1 周だけ通して、画面下のリストを上から **○ / × / 違和感** でチェック。
> 完了したら違和感だけ Claude に戻す。

URL：https://rainybrainch.github.io/pomojikan/

---

## ⏱ 起動〜開始（最初の 30 秒）

- [ ] 開いたら背景が **桜の紫**（自動＝春・5 月）
- [ ] ☰ が右上にある
- [ ] アップデート後なら 🎓 ツアー（5 ページ）が自動で出る → 流し読みで閉じる
- [ ] timer-text に `25:00` 表示
- [ ] 時計の縁を **短タップ** → 分数が変わる（試したら 25 に戻して）
- [ ] 時計の縁を **長押し** → 紫色（休憩モード）に変化

## 🎯 作業開始（▶ ボタン）

- [ ] ▶ 始める を押すと **🐦 カッコウ** が鳴る
- [ ] 通知許可ダイアログ（初回のみ）→ 許可
- [ ] HUD（左上）が出る → 「★リーダー Lv / ⚡EXP / 💡推薦」
- [ ] 字が降ってくる
- [ ] progress-pill に「🔁 1/3セット ・ 🎯 作業中」

## 🎰 物理（作業中 5 分くらい観察）

- [ ] 字が中央の棚に積もる
- [ ] 棚から押し出された字が **下端から落ちて EXP 化**（+XP 表示）
- [ ] 字をドラッグできる
- [ ] 同じ字をドラッグして重ねると融合（光る）
- [ ] 30 個以上字があっても **画面が重くならない**

## ⚡ パーティ / 効果

- [ ] パーティ下に **現効果パネル** ── 3 グループ（成長／物理／状態）×3 セル
- [ ] 強効果セルは金色光彩
- [ ] パッシブが「⚙ N/20」（春の祝福を含むので最低 1）
- [ ] 図鑑で熟語タップ → 物語・効果プレビュー・✨ ワンタップ編成ボタン
- [ ] コンボ発動の瞬間、**画面に色の aura が 3 秒滲む**（タグ次第で緑/赤/青/紫）

## 🫧 休憩遷移（25 分後 / 早く確認したいなら歯車で 1 分に）

- [ ] 25 分経過 → flashCompletionBurst「☔ 凝縮 完了」
- [ ] 0.6 秒後に休憩開始 → **🐦 カッコウ**
- [ ] 背景／HUDが「🫧 休憩中」に
- [ ] 字が**泡となって上昇** → リーダーが育つ
- [ ] OS 通知「🫧 作業完了 ── 休憩へ」が出る（許可済なら）

## 🔁 自動継続（5 分後）

- [ ] 休憩完了 → 1.2 秒後に **自動で次の作業へ**（手動で ▶ 不要）
- [ ] **🐦 カッコウ** 再生
- [ ] progress-pill が「🔁 2/3セット」に

## 🎨 テーマ切替

- [ ] メニュー → 🎨 テーマ → 「夏」を選ぶ → 背景が深青に滑らかフェード
- [ ] 「ダーク」「ライト」も試す → 紙色も含めて変化
- [ ] 「自動」に戻して終了

## 📺 画面外（Chrome デスクトップなら）

- [ ] メニュー → 📺 小窓タイマー → 260x260 小窓が出る
- [ ] 他アプリ最前面でも小窓が見える
- [ ] もう一度押すと閉じる

## 🛡 異常系（軽く）

- [ ] 作業中に ↺ 停止 → idle に戻る、カッコウは鳴らない
- [ ] 通知拒否しても破綻しない
- [ ] スマホで時計の縁ドラッグが指追従

---

## 戻ってきて報告するときのテンプレ

```
✓ 大部分 OK
✗ NG / 違和感：
  - ○○ がスマホで小さい
  - ○○ の音量がうるさい
  - 季節 ○○ の色味が地味
  - ○○ が押しにくい
  - ○○ が予期しないタイミングで起きた
```

これ書いて貼ってもらえれば 1 サイクルで詰めます。
