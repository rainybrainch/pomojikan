# 🌑 ぽもじかん（Pomojikan）

文字が降ってくる作業タイマー。ポモドーロのリズムで文字を積み上げ、図鑑で集める。
育成・コレクション・物理エンジン搭載の PWA。

[🌧 開く ── https://rainybrainch.github.io/pomojikan/](https://rainybrainch.github.io/pomojikan/)

## 特徴

- 📅 **ポモドーロタイマー**：25/5、50/10、15/3、自由カスタム
- 🌧 **文字が降る**：作業中＝徳のしずく／休憩中＝罪の泡
- 🧲 **物理エンジン**：重力・反発・摩擦・空間グリッド衝突
- ⚛ **物理スライダー**：重力/反発/摩擦/出現間隔をリアルタイム調整
- 👆 **物理ドラッグ**：指で掴んで投げられる（フリック慣性付き）
- 🌟 **スイカ合体**：同じ文字どうしで「大→真→極」3段階進化
- 📚 **ぽもじ図鑑**：Lv 9 までの育成・コレクション
- ✨🌈 **レア文字**：初出時に金 1%、虹 0.2% で出現
- 🏆 **17 達成バッジ**：完走/連続/図鑑/レア/合体/Lv9
- 🔥 **連続日数**：毎日完走で +1、最高記録保持
- 🎯 **デイリーチャレンジ**：日替わりお題
- 📊 **統計**：7日棒グラフ + 30日ヒートマップ + 時間帯ヒートマップ
- 🎨 **5 テーマ**（罪徳/雨/砂/雪/桜）+ カラーパレットカスタム
- 🖋 **3 フォント**（明朝/丸ゴシック/角ゴシック）
- 🌊 **6 BGM**（雨/激雨/夜雨/風/波/無音・Web Audio）
- 📜 **4 言葉モード**：七大罪徳 / 自由 / 服牢365 / 混在
- 🌍 **多言語**：日/英 切替
- ⛶ **フォーカスモード**：UI 最小化＋フルスクリーン
- 💾 **設定プリセット**：複数保存・呼び出し
- 📤 **データ書出**：JSON / CSV / シェア（Web Share API）
- 📱 **PWA**：オフライン動作・ホーム画面追加・通知
