# 🌑 ぽもじかん（Pomojikan）

> **字が降る。集中が、主人公を選ぶ。**

文字育成ポモドーロ ── 集中の時間に字が降り、休憩の時間に泡となって主人公を育てる PWA。

[🌧 開く ── https://rainybrainch.github.io/pomojikan/](https://rainybrainch.github.io/pomojikan/)

---

## 哲学 ── 重力と引力

この字は、ユーザーのためではなく、重力のために存在する。

集中とは、内側に質量を溜めることだ。25 分のあいだ、字が降り積もる。  
休憩とは、溜まった質量を泡に変えて引力で空に返すことだ。5 分のあいだ、字が浮上して主人公の経験値になる。

ぽもじかんはこの **重力（凝縮・作業）と引力（発散・休憩）** の物理法則を、画面に直接刻んだポモドーロタイマーである。雨は落ちる、泡は昇る。それだけのことを、4 体のパーティと育成ゲームで支えている。

タイマーは、終わりを置く装置だ。だからこのアプリには、ランキングも、順位も、連勝記録も、復帰を強制する通知もない。

## システム

| 要素 | 役割 |
|---|---|
| **主人公**（1 字） | 最初に選び、好きな名前を付ける。消滅不可・最も成長する核 |
| **仲間**（最大 3 字） | 主人公とパーティを組む。消滅不可・主人公と並行して育つ |
| **降る字** | 集中中は字、休憩中は泡。タップで消滅 → 4 体に EXP 配分 |
| **★1〜★6** | 平均 Lv が上がるたびにレア帯が解放される（段階解放） |
| **書体進化** | 字が Lv 10／30／70 で楷書 → 行書 → 草書に進化 |
| **特性スタッキング** | 字のタグから固有パーク（20 種以上）が付き、4 体ぶん同時に積まれる |
| **同質共振** | 同じタグの字を融合すると XP+50% ／ タグ系パークがあるとさらに乗算 |

レア帯は **文字種で世界が広がる** 10 段階構造（v2 / 2026-05-16〜）：

```
★1  始まりの音       (Lv 1〜)   ── ひらがな 81 字
★2  もうひとつの音   (Lv 3〜)   ── カタカナ 82 字
★3  量と順序         (Lv 6〜)   ── 数字 34 字（半角・漢数字・ローマ数字）
★4  異邦の文字       (Lv 10〜)  ── 英語 52 字（A-Z / a-z）
★5  学びの初日       (Lv 15〜)  ── 拾級漢字 151 字
★6  日常の漢字       (Lv 22〜)  ── 五級漢字 208 字
★7  使い慣れた漢字   (Lv 30〜)  ── 三級漢字 137 字
★8  深まりの漢字     (Lv 45〜)  ── 一級漢字 97 字
★9  美と古典         (Lv 65〜)  ── 初段漢字 13 字
★10 七徳七大罪の領域 (Lv 100〜) ── 拾段漢字 19 字（神字）
```

将来は ★100 まで段階を増やせる設計。古字／時代別流行語／古典外国語などを順次解放予定。

## 数値

- **字** ── **874 字**（★1=81 / ★2=82 / ★3=34 / ★4=52 / ★5=151 / ★6=208 / ★7=137 / ★8=97 / ★9=13 / ★10=19）
- **熟語レシピ** ── **246 個**（同じ字どうしの衝突で合成判定・★5 以上の漢字熟語）
- **パーティ** ── **最大 4 体**（主人公 1 + 仲間 3）
- **タグ** ── 七徳・七大罪・感情・時・禅・神字・武・学・自然・植物・美・宗教・仏教・思想 ほか
- **タイマープリセット** ── 25/5 ・ 50/10 ・ 15/3 ・ カスタム秒
- **データ** ── 端末内のみ（`localStorage`：キー `pomojikan_v30`）
- **PWA** ── オフライン動作 / ホーム画面追加 / マニフェスト・サービスワーカー完備

レア字は落下速度がゆっくりになる（最大 0.6 倍）。重力と物語を、速度で描いている。

## 接続 ── この字はどこから来たか

ぽもじかんで降る字は、製作者が日々選び抜いた語録から生まれている。七徳・七大罪・感情・神字といったタグ群は、別ラインで動く名言プロジェクトと地続きで、同じ土から生えている。だから降る字は、ただの辞書項目ではなく、**意味の重さを持った字** である。

主人公を選ぶ瞬間、その字はあなたの語録に変わる。

## 技術

- **単一 HTML + Vanilla JS + localStorage** ── サーバー送信ゼロ、アクセス解析なし
- **PWA** ── `manifest.json` / `sw.js` / マスカブルアイコン / Share Target / Shortcuts（25分/15分/45分）
- **物理エンジン** ── 重力・反発・摩擦・空間グリッド衝突・物理ドラッグ
- **オンボーディング** ── 初回起動時に 4 ステップで世界観を案内
- **引継ぎコード** ── ユーザー ID（P-XXXX-XXXX 形式）方式・別端末復元可
- **バックグラウンド対応** ── アプリを閉じてもタイマー続行、復帰時にオフラインボーナス字が降る
