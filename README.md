# 🌑 ぽもじかん（Pomojikan）── v2.9

漢字・古典・神話・科学… **22,160 種類**の言葉が降る百科事典級のポモドーロ作業タイマー。
**🎭 パーティ 4 体を育てて ☆ レアリティを段階解放** する PWA 育成ゲーム。

[🌧 開く ── https://rainybrainch.github.io/pomojikan/](https://rainybrainch.github.io/pomojikan/)

## 🆕 v2.9 リニューアル（2026-05-12）

ぽもじかんが「文字育成ゲーム」に昇華:

- **🎭 パーティシステム**：4 体のぽもじを選んで主人公 1 + 仲間 3 として育てる。優先的に降り、消滅不可
- **☆ レアリティ段階解放**：☆1〜☆5 が classLevel に応じて開放（☆3=6級/☆4=3級/☆5=1級）
  - 序盤でも 0.1% の確率で ☆3 が降る「千分の一の奇跡」
- **🌧 雨ぽもじ / 🫧 泡ぽもじ**：22,160 種を 4:1 で分割（集中=学び/休憩=遊び癒し）。独立した解放進捗
- **消滅 EXP**：不要なぽもじをダブルタップで消滅 → パーティ 4 体に均等配分
- **カスタム時間 1 秒単位**：集中・休憩の時間を分・秒で自由設定
- **バックグラウンド復帰演出**：5 秒以上経過時に「☂ N 粒のぽもじが やってきます」段階配信
- **96 カテゴリ × 18 大分類**：図鑑をフィルタで俯瞰

- 擬音語擬態語 ／ 歴史人物・時代 ／ 医療（病名・解剖・薬学・診療科）
- 現代（テック・ビジネス・カルチャー）／ 世界（国・首都・観光地・地理）／ 法経政
- 科学（物理・化学・数学・単位）／ 料理（食材・タンパク質・調理法）
- サブカル（アニメ・ゲーム・サブカル）／ 音楽（クラシック・ポップ・楽派・邦楽）

## レア発見演出（v2.3〜）

- 🌈 虹：画面全体に虹色ラジアル(1.8s, hue-rotate) + 18 粒パーティクル + 5音アルペジオ + 高音きらめき + 振動 7段

- レア文字は物理サイズも 1.08〜1.25 倍で大きく落ちる（v2.5〜）
