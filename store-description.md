## 長い説明（4000 字以内）

🌧 **ぽもじかん** は、ポモドーロ・タイマーと文字育成ゲームを合わせた集中アプリです。

集中する時間に **「字」（ぽもじ）が画面に降ってきて**、休憩中はそれらが **泡となって浮上し、リーダーの経験値**になります。「凝縮（作業）と発散（休憩）」のリズムを物理で可視化しました。

### 🎮 遊び方
1. リーダー（1 字）を選ぶ ── 最初は **ひらがな** から、リロール可能
2. タイマー円の **縁をなぞって時間を設定**（短タップ／ドラッグ／長押しで休憩時間）
3. ▶ 開始 → 字が降ってくる ── ドラッグで重ねて融合・タップで消滅・経験値化
4. 休憩中、貯まった字が泡となって上昇し、リーダーを育てる
5. 同じ字を合成して進化、平均 Lv で新しいレアリティが解放される

### ✨ 特徴

**字とコレクション**
- **41,890 字**（ひらがな・カタカナ・漢字・ハングル・古代文字・世界 17 系統）
- **4,546 種の熟語コンボ**、すべてに固有効果と物語
- **★1〜★16 の段階解放**（文字種ベース）、最初は「ひらがなだけ」のシンプルな世界から
- **書体変化** ── 字が成長段階で楷書 → 行書 → 草書、計 15 段階の進化（Lv 1,000,000 まで）

**パーティと育成**
- **★ リーダー制** ── 図鑑から字をタップ →「★ リーダーに設定」で一発切替
- **特性スタッキング** ── 字の意味から固有特性、4 体ぶん同時に働く
- **🗂 パーティ プリセット** ── 編成を 12 個まで名前付きで保存・ワンタップ切替
- **⚡ 現効果パネル** ── EXP・重力・融合範囲・粒数・進化加速が一目で見える

**コンボの 3 層構造**
- **特性**：パーティ字の固有・育つ（強）
- **コンボ**：熟語成立で発動・物語のある効果（中〜強）
- **⚙ パッシブ 16 種**：コレクション／継続マイルストーンの恒久弱効果

**コインプッシャー型物理**
- 中央 76% が棚、両端 12% は穴
- 字が押されて棚を外れると落下 → 自動 EXP 化
- 字が累積しない設計＝何時間遊んでも軽快

**🎓 達成バッジ 21 個**
- 連続日数・累計サイクル・字発見・熟語解放・リーダー Lv の長期目標
- 達成タップで当時のセレモニーが再生される

### 🔄 バックグラウンド・画面外対応
- アプリを閉じても **タイマーは続行**
- 復帰時に「オフラインボーナス」字（50%）が降る
- 💡 **Wake Lock**：作業中は画面スリープ抑制
- 🔔 **通知**：サイクル完了を OS 通知（許可制）
- 📺 **Picture-in-Picture**：他アプリ作業中も小窓タイマー（Chrome/Edge デスクトップ）

### 💾 データ管理
- **localStorage 単独**、サーバー通信ゼロ
- メニュー「💾 データ管理」から全データを JSON で書き出し／復元
- 別端末への引き継ぎも 3 ステップ

### 📖 図鑑コレクション
- 41,890 字 / 4,546 熟語 / 21 バッジを「ぽも字図鑑」で確認
- ★ お気に入り、進捗率バッジ、フィルタの「🎯 1 行表示」
- 熟語詳細：物語・効果プレビュー数値・「✨ このコンボで編成」ワンタップ

### 🌧 設計思想
- ランキング・順位・連勝記録などの **滞留促進系の仕掛けは入れません**
- 「凝縮の時間」と「発散の時間」を分け、後者には**終わり**を置きます
- 通知でアプリへの復帰を強制しません

### 🔒 プライバシー
- データは **すべてあなたの端末内のみ**（localStorage キー `pomojikan_v30`）
- サーバーへの送信・トラッキング・広告は **一切ありません**
- アカウント作成・メール登録 **不要**
- データ書き出しを使う時のみ、JSON ファイルを自分で別端末に渡す形式

### こんな人におすすめ
- ポモドーロを続けたいけど、ただのタイマーは飽きてしまう
- 集中時間を「目に見える形」で残したい
- 字や言葉が好き／漢字が好き／世界中の文字を集めたい
- 静かで邪魔をしないアプリを探している
- 何十年でも遊べる育成型がいい（長期達成バッジ・進化15段階）

## データセーフティ申告（Play Console 用）

### 収集データ：**なし**
- ユーザーがアプリに入力する情報は、すべて端末内（localStorage）にのみ保存
- サーバー通信は GitHub Pages からの静的ファイル取得・Google Fonts のみ
- アナリティクス・広告・トラッキング ID 一切なし

### 共有データ：**なし**

### 通知の利用
- タイマー完了時にローカル通知（Notifications API）を表示
- 通知内容は端末内で生成、外部送信なし
- 許可制（拒否しても全機能動作）

### Wake Lock の利用
- 作業中／休憩中のみ画面スリープを抑制
- 停止時に自動解放
- ブラウザ標準 API

### Picture-in-Picture の利用
- ユーザーがメニュー「📺 小窓タイマー」を明示的にタップした時のみ起動
- Chrome/Edge デスクトップ限定（他環境では非表示）

### データ暗号化：該当なし（端末内ローカル保存）

### データ削除：**設定 → 📊 記録 → 危険な操作 → 全データを消去**から可能
