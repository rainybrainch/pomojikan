## 長い説明（4000 字以内）

🌧 **ぽもじかん** は、ポモドーロ・タイマーと文字育成ゲームを合わせた集中アプリです。

集中する時間に **「字」（ぽもじ）が画面に降ってきて**、休憩中はそれらが **泡となって浮上し、主人公の経験値**になります。「凝縮（作業）と発散（休憩）」のリズムを物理で可視化しました。

### 🎮 遊び方
1. 主人公（1 字）を選ぶ ── 最初は **ひらがな** から、リロール可能
2. 集中タイマーを開始（25 分／50 分／15 分／自由設定）
3. 10 秒ごとに字が降ってくる ── ドラッグで重ねて融合・タップで消滅・経験値化
4. 休憩中、貯まった字が泡となって上昇し、主人公を育てる
5. 同じ字を合成して進化、平均 Lv で新しいレアリティが解放される

### ✨ 特徴
- **特性スタッキング**（ヴァンパイアサバイバー方式）── 字の意味から自動で固有特性が付き、4 体ぶん同時に働く
- **★1〜★10 の段階解放**（文字種ベース）── ひらがな → カタカナ → 数字 → 英語 → 漢字（拾級→拾段）と段階的に世界が広がる。最初は「ひらがなだけ」のシンプルな世界から始まる
- **書体変化** ── 字が Lv 10／30／70 で楷書 → 行書 → 草書と進化
- **同質共振** ── 同じタグ（七徳・七大罪・感情・水・火など）の字を融合すると XP+50%
- **新発見バッジ** ── 初めて見る字に「新」金マーク
- **背景の雨と泡** ── 集中中は雨、休憩中は泡が画面全体に
- **音響** ── 雨音・泡音はオン／オフ可（合成音、ファイル不要）

### 🔄 バックグラウンド対応
アプリを閉じても **タイマーは続行**。復帰時に、隠れていた時間ぶんの「オフラインボーナス」字（50%）が一気に降下します。

### 🔑 引継ぎコード（一般ゲーム方式）
- 初回プレイ時に **ユーザー ID（P-XXXX-XXXX）** を自動発行
- 「引継ぎコードを発行」で別端末にデータ移行可能
- お問い合わせ時はユーザー ID を提示

### 📖 図鑑コレクション
- 874 字（ひらがな・カタカナ・数字・英語・漢字 全段階）を「ぽも字図鑑」で確認
- ★1〜★10 のレアリティ別タブで絞り込み
- 字をタップすると意味・タグ・出現回数・関連熟語が表示
- 246 種の熟語レシピを内包（★5 以上の漢字熟語）

### 🛡 パーティ共有
- 自分のパーティ設定（4 字＋特性）を URL で共有可能
- 受け取った人は「プレビューモード」で試すだけ（自分の進捗には影響しない）

### 🌧 設計思想
- ランキング・順位・連勝記録などの **滞留促進系の仕掛けは入れません**
- 「凝縮の時間」と「発散の時間」を分け、後者には**終わり**を置きます
- 通知でアプリへの復帰を強制しません

### 🔒 プライバシー
- データは **すべてあなたの端末内のみ**（localStorage）に保存されます
- サーバーへの送信・トラッキング・広告は **一切ありません**
- アカウント作成・メール登録 **不要**
- 引継ぎコードを使う時のみ、コードに含まれる自分のプレイデータを自分で別端末に渡す形式

### こんな人におすすめ
- ポモドーロを続けたいけど、ただのタイマーは飽きてしまう
- 集中時間を「目に見える形」で残したい
- 字や言葉が好き／漢字が好き
- 静かで邪魔をしないアプリを探している
